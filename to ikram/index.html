IROgame : The positive effect game.

there's a difference between positive infulence games and regular games.

We created, organized and designed this game for all ages but specially kids and youth to teach them how to save environment by throwing garbage in the garbage can and not somewhere else, we carefully choose this idea as we saw that Moroccan youth are in need of fun education so by playing this game they will accept it and apply the rules of the game in the daily life, the game is simple but in the same time it has a powerful idea.
